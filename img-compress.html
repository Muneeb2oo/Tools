Frequently Asked Questions

Is this tool really free to use? +

Yes, our PDF and image compression tool is completely free to use with no hidden charges. You can compress as many files as you need without any limitations.

Are my files secure when using this tool? +

Absolutely! All file processing happens directly in your browser. Your files are never uploaded to any server, ensuring complete privacy and security.

What file formats are supported? +

For images, we support JPG, PNG, GIF, and WebP formats. For documents, we support PDF files. We're continuously working to add support for more file types.

What's the maximum file size I can compress? +

You can compress images up to 50MB each and PDFs up to 100MB. These limits are set to ensure good performance in your browser.

How much can I reduce my file sizes? +

Compression results vary depending on the file type and content. Typically, you can expect:

  • JPG images: 60-80% reduction
  • PNG images: 40-70% reduction
  • PDF documents: 30-90% reduction
